简单来说，模运算具有如下一些运算性质：
1. ( a + b) % c = ( ( a % c ) + ( b % c ) ) % c
2. ( a * b) % c = ( ( a % c ) * ( b % c ) ) % c
3. ( a – b) % c = ( ( a % c ) – ( b % c ) ) % c
最基本的操作就是通过取余数避开溢出而达到正确结果

因此可以在不做可能导致溢出的大数运算的条件下，实现验证答案的正确性（同余即可判定正确）
